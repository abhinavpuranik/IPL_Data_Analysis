# IPL Dataset Analysis using NumPy and Pandas

A data analysis project based on the Indian Premier League (IPL) datasets using Python, NumPy, Pandas, and Matplotlib. The project focuses on extracting insights from match and ball-by-ball delivery data through data cleaning, aggregation, statistical analysis, and visualization.

## Datasets Used

- `matches.csv`
- `deliveries.csv`

These datasets contain detailed IPL match information including teams, players, runs, wickets, overs, seasons, and match results.

## Features

- Match-level and season-level analysis
- Runs scored per over
- Team-wise performance analysis
- Batter strike rate analysis
- Bowler economy calculations
- Season comparisons
- Data cleaning and preprocessing
- Statistical aggregations using NumPy and Pandas
- Visualizations using Matplotlib
